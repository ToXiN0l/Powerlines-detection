# Powerlines Detection

Веб-сервис на базе FastAPI и ultralytics для детекции линий электропередач на изображениях. Сейчас веб-приложение развернуто на Yandex cloud и может работать по ссылке http://89.169.165.186

## Запуск через Docker

1. Соберите образ:
```bash
docker build -t powerlines-app .
```
2. Запустите контейнер:
```bash
docker run -d -p 80:8000 --name powerlines-container powerlines-app
```
после флага --name может идти любое название контейнера.

## Использование

После запуска сервис доступен по внешнему IP-адресу сервера: http://<ваш ip> (используется порт 80).
Интерактивная документация API (Swagger) находится по адресу: http://<ваш ip>/docs.
